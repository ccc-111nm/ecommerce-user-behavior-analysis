# 电商用户行为多维分析系统
基于Spark SQL分布式计算框架，对1万条电商用户购买数据进行多维度业务分析，输出可视化看板，支撑运营决策。

## 技术栈
- 计算引擎：Spark 3.5.4 + Scala 2.12
- 数据存储：MySQL 8.0
- 数据可视化：FineBI
- 运行环境：Hadoop YARN

## 核心分析指标
1. 用户购买力分析：统计每个用户的累计购买商品总量，定位高价值用户
2. 商品品类分析：统计各品类的总销量、购买次数，分析品类表现
3. 地区销售分析：分城市统计各品类的销售数据，对比地域差异
4. 销量趋势分析：按时间维度统计商品销量变化，识别销售规律
5. Top用户排行：筛选购买量最高的Top10用户，做精细化运营

## 项目成果
- 完成1万条模拟电商数据的生成、清洗与入库
- 实现5个核心业务维度的SQL统计分析，结果准确可复用
- 搭建FineBI可视化看板，直观展示销售数据与用户特征
- 掌握分布式数据处理流程，具备业务数据指标拆解能力
